# 🌱 AgroVision AI

Sistema de monitoramento inteligente com detecção de objetos em tempo real utilizando **FastAPI**, **OpenCV** e **YOLO (Ultralytics)**.

---

## 🚀 Visão geral

O AgroVision AI captura imagens de uma câmera ou vídeo, aplica um modelo de visão computacional (YOLO) para detecção de objetos e:

- Exibe o monitoramento em tempo real
- Registra eventos detectados
- Salva evidências (imagens)
- Armazena dados em banco SQLite
- Apresenta um dashboard web moderno

---

## 🧠 Tecnologias utilizadas

- FastAPI
- Uvicorn
- OpenCV
- Ultralytics (YOLO)
- SQLite
- Jinja2
- HTML + CSS (frontend customizado)
